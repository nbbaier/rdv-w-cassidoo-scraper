## Interview question of the week

This week's question:\
**Given an integer `n`, return `true` if it's a perfect square AND when reversed, is still a perfect square.**

Example:

```
> reversedSquares(9)
> true

> reversedSquares(441)
> true

> reversedSquares(25)
> false
```
