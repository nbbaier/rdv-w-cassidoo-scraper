## Interview question of the week

**Given a list of menu items and their prices, and the amount in your wallet, return all combinations of items that you can buy.**

Example:

```
menu = { 'taco' : 1, 'burger' : 2, 'shawarma' : 3 }
wallet = 3
> combo(menu, wallet)
['taco', 'taco', 'taco']
['taco', 'burger']
['shawarma']
```
