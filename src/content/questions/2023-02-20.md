## Interview question of the week

This week’s question:\
**Given a string of parenthesis, return the number of parenthesis you need to add to the string in order for it to be balanced.**

Examples:

```
> numBalanced(\`()\`)
> 0

> numBalanced(\`(()\`)
> 1

> numBalanced(\`))()))))()\`)
> 6

> numBalanced(\`)))))\`)
> 5
```
