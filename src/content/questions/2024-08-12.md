This week's question:

**Given an array of logs and variable assignments, return a list of all unused variables.**

Examples:

```
> findUnused(["a = 1", "b = a", "c = 2", "log(b)"]);
> ["c"]

> findUnused(["a = 1", "b = a", "c = 2", "log(c)"]);
> ["a", "b"]
```
