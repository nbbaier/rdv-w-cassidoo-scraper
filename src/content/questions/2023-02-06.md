This week's question:\
**Spreadsheet programs often use the A1 Reference Style to refer to columns. Given a column name in this style, return its column number.**

Examples of column names to their numbers:

```
A -> 1
B -> 2
C -> 3
// etc
Z -> 26
AA -> 27
AB -> 28 
// etc
AAA -> 703
```
