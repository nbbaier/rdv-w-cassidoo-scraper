## Interview question of the week

This week’s question:\
**Write a function to find the longest common prefix string in an array of strings.**

Example:

```
$ longestPrefix(["cranberry","crawfish","crap"])
$ "cra"

$ longestPrefix(["parrot", "poodle", "fish"])
$ ""
```
