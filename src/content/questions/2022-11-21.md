## Interview question of the week

This week’s question:\
**Write a function that takes a string of slashes (`\` and `/`) and returns all of those slashes drawn downwards in a line connecting them.**

Example:

```
$ verticalSlashes('\\\//\/\\')
\
 \
  \
  /
 /
 \
 /
 \
  \
$ verticalSlashes('\\\\')
\
 \
  \
   \
```
