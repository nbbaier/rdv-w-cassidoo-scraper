## Interview question of the week

This week's question:\
**Write a function that draws an ASCII art cube of given height `n`.**

Example:

```
> drawCube(2)
"
  +----+
 /    /|
+----+ |
|    | +
|    |/
+----+
"

> drawCube(4)
"
   +--------+
  /        /|
 /        / |
+--------+  |
|        |  |
|        |  +
|        | /
|        |/
+--------+
"
```
