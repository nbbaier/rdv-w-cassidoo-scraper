## Interview question of the week

**Write a function addg that adds from many invocations until it sees an empty invocation.**

```
addg()  // undefined
  addg(2)()  // 2
  addg(2)(7)()  // 9
  addg(3)(4)(0)()  // 7
  addg(1)(2)(4)(8)()  // 15
```
