This week's question:\
**Given an integer `n`, return true if `n^3` and `n` have the same set of digits.**

Example:

```
$ sameDigits(1) // true
$ sameDigits(10) // true
$ sameDigits(251894) // true
$ sameDigits(251895) // false
```
